# Pet Shop do Vargas

Este projeto foi desenvolvido como parte da disciplina **Fundamentos de Sistemas Web**.

O site apresenta informações sobre produtos e serviços de um pet shop, incluindo imagens, descrições e valores, além de permitir que o usuário realize o agendamento de serviços para seu pet.


## Tecnologias utilizadas
- HTML5
- CSS3
- Bootstrap
- JavaScript

## Funcionalidades
- Visualização de categorias de produtos
- Exibição de produtos com imagem, descrição e valor
- Apresentação de serviços do pet shop
- Carrossel de imagens na página inicial
- Formulário de cadastro do cliente e do pet
- Agendamento de serviços (banho ou tosa)
- Escolha de data e horário para atendimento
- Confirmação do agendamento com JavaScript

## Acessibilidade

O site utiliza o atributo `alt` nas imagens para auxiliar usuários que utilizam leitores de tela.
